Hyperelastic
============

*Constitutive hyperelastic material formulations for* https://github.com/adtzlr/felupe .

This package provides the essential building blocks for constitutive hyperelastic material formulations. This includes material behaviour-independent :mod:`Spaces <.spaces>` and :mod:`Frameworks <.frameworks>` as well as material behaviour-dependent :mod:`Models <.models>`.
