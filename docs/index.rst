Hyperelastic
============

*Constitutive hyperelastic material formulations for FElupe.*

This package provides the essential building blocks for constitutive hyperelastic material formulations. This includes material behaviour-independent :mod:`.spaces` and :mod:`.frameworks` as well as material behaviour-dependent :mod:`.models`.
